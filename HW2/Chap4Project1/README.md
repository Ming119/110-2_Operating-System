### Build

To compile the source files

```bash
make
```

To clean the object and binary files

```bash
make clean
```

Using the solution provide by the text-book

```bash
./bin/main
```

Random generate number

```bash
./bin/main random
```

Manually input data

```bash
./bin/main custom
```
